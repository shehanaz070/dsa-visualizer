What is a Tree?

A tree is a hierarchical data structure consisting of nodes, with a single root node and potentially many levels of additional nodes.

Types of Trees

Binary Tree: Each node has at most 2 children.
Binary Search Tree (BST): Left < Root < Right
AVL Tree: Self-balancing BST
Red-Black Tree: Balanced BST with coloring rules
Complete / Full / Perfect Binary Tree
Trie: Tree used to store strings (prefix tree)
